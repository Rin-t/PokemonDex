#  PokemonDex

ポケットモンスター。縮めてポケモン。この星の不思議な不思議な生き物。
森に、空に、海に世界の至る所でその姿を見ることができる。
そんなポケモンの姿をまとめた図鑑。そう、それがPokemonDex。
これからもちょっとずつ機能を拡張していきます！！
